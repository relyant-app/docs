title: "Add Images and Files"
description: "Attach photos, manuals, and documents to an asset record"
---

<Note>
  **Required role:** **Admin** or **Manager**
</Note>

You can attach images and documents to any asset â€” useful for photos of the equipment, manufacturer manuals, warranty documents, or inspection reports.

## Steps

<Steps>
  <Step title="Open the asset">
    In the left sidebar, click **Assets**, then click the asset you want to update.
  </Step>
  <Step title="Go to the Files tab">
    On the asset detail page, click the **Files** or **Attachments** tab.
  </Step>
  <Step title="Upload a file">
    Click **Upload** and select the file from your device. Supported formats include:
    - Images: JPG, PNG, GIF, WebP
    - Documents: PDF, Word, Excel
    - Other: any file up to the size limit
  </Step>
  <Step title="Add a description (optional)">
    Add a label or description to help identify the file (e.g., "Installation Manual", "Warranty Certificate").
  </Step>
  <Step title="Save">
    The file is automatically saved and attached to the asset. It will appear in the Files tab for any member with access to view the asset.
  </Step>
</Steps>
